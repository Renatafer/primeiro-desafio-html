HTML Não é uma linguagem de programação

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas.

Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, atributos etc..

Estrutura básica de um HTML

HTML = <> Abre e fecha , para o arquivo saber que é uma estrutura HTML.

HEAD = <> Abre e fecha , a cabeça do arquivo html , tudo que que carrega antes de exibir para os usuários. EXP: título ou title.

BODY = <> Abre e fecha , o corpo do arquivo html , tudo que estiver no corpo aparecerá para o usúrio.

Tags básicas de texto HTML

TAGS
  1. TAG: H1 até H6: Titulos da pág do mais importante para menos importante.
  2. TAG: P: Para sinalizar parágrafos.
  3. TAG: U: Serve para sublinhar.
  4. TAG: MARK: Serve para marcação de texto.
  5. TAG: BLOCKQUOTE: Serve para fazer uma citação, de pensamento ou uma autoria.
  6. TAG: SUP: Serve para alinhar um elemento ou texto em cima.
  7. TAG: I: Deixa em itálico.
  8. TAG: Strong: Deixa em negrito e da ênfase a palavra.
  9. TAG: Input : Cria um campo para , digitação no navegador.
  10. TAG :Img: Serve para inserir uma imagem junto com o atributo src.

Atributos das tags

São atributos que uma tag pode ter.

Atributos Genéricos e Específicos

Genérico : Eles variam e podem conter atributos diversos. EXP: CLASSE E ID

Específicos Eles por sua vez recebem atributos específicos. EXP:TYPE NUMBER: Só recebe numero, TYPE NOME: Só recebe letra.

Listas Ordenadas e não Ordenadas

UL: São listas não Ordenadas: Ela coloca marcadores antes de cada elemento.

OL: São listas Ordenadas: Ela enumera os elementos em sequência.
